## Logos
I provide logos for each exchange in various formats:
* SVG
* PNG 80x40px
* PNG 160x80px
* PNG 320x160px
* PNG 640x320px

All logos are either the originals in vector format or result from a personal vectorization made from the original pixelated logo.

Missing logos will only be provided once a vectorization closer to the original has been done.
